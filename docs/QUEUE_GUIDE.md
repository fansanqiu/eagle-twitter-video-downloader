# 下载队列使用指南 / Download Queue Guide

## 中文

### 快速开始

1. **添加视频到队列**
   - 复制任意支持网站的视频链接
   - 粘贴到底部输入框
   - 点击"下载"按钮

2. **批量下载**
   - 可以连续添加多个视频
   - 插件会自动管理下载队列
   - 最多同时下载 3 个视频

3. **查看进度**
   - 每个视频显示独立的进度条
   - 实时显示下载速度和剩余时间
   - 完成后自动导入到 Eagle

4. **处理错误**
   - 失败的下载会显示错误信息
   - 点击"重试"按钮重新下载
   - 错误不会影响其他下载

### 下载状态说明

- **等待中**: 视频在队列中等待下载槽位
- **准备中**: 正在获取视频信息
- **下载中**: 正在下载视频文件
- **完成**: 已下载并导入到 Eagle
- **错误**: 下载失败，可以重试

### 下载历史

- 已完成的下载会保存在历史记录中
- 关闭插件后重新打开可以看到历史
- 最多保存最近 50 个下载记录

## English

### Quick Start

1. **Add Videos to Queue**
   - Copy a video URL from any supported website
   - Paste it into the input field at the bottom
   - Click the "Download" button

2. **Batch Downloads**
   - Add multiple videos consecutively
   - The plugin automatically manages the download queue
   - Up to 3 videos download simultaneously

3. **View Progress**
   - Each video shows an individual progress bar
   - Real-time display of download speed and remaining time
   - Automatically imports to Eagle when complete

4. **Handle Errors**
   - Failed downloads show error messages
   - Click the "Retry" button to re-download
   - Errors don't affect other downloads

### Download States

- **Waiting**: Video is queued waiting for a download slot
- **Preparing**: Fetching video information
- **Downloading**: Downloading video file
- **Complete**: Downloaded and imported to Eagle
- **Error**: Download failed, can retry

### Download History

- Completed downloads are saved in history
- History persists after closing and reopening the plugin
- Stores up to 50 most recent downloads

## Tips / 使用技巧

### 中文

- 可以在下载进行时继续添加新视频
- 队列会自动滚动，最新添加的视频在顶部
- 如果遇到网络问题，可以稍后重试失败的下载

### English

- History helps you track downloaded videos

## Troubleshooting / 故障排除

### 下载失败 / Download Failed

**问题**: 视频显示错误状态
**解决方案**:

1. 检查网络连接
2. 确认视频链接有效
3. 点击"重试"按钮
4. 如果持续失败，尝试复制新的链接

### 进度卡住 / Progress Stuck

**问题**: 进度条长时间不动
**解决方案**:

1. 等待几分钟（大文件需要时间）
2. 检查网络速度
3. 如果超过 10 分钟无响应，关闭插件重新打开

### Eagle 导入失败 / Eagle Import Failed

**问题**: 下载完成但未导入 Eagle
**解决方案**:

1. 确认 Eagle 正在运行
2. 检查 Eagle 库是否有足够空间
3. 视频文件已下载到临时目录，可以手动导入
